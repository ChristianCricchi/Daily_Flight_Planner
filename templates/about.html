Welcome to Daily Flight Planner!

Daily Flight Planner is an innovative application designed to digitise and streamline the daily dispatch planning process for airports. Our platform offers a user-friendly interface that enhances efficiency and accuracy in managing flight operations.

  1. Log In: There are two types of logins: AdminOps and Dispatcher, each with access to different functionalities. To access, you need a login created by Adminops.
      AdminOps:It is a privileged access. It can access the profile currently in use, access the home page, search the database, edit and delete flights. It can access the about section. It has access to the dispatch section where it can modify and delete dispatcher. It has access to the register section where it can create a new account. It can access the new flight section where it can create a new flight.
      Dispatcher:It can access the Profile page to view the currently logged-in profile. It can access the Home page to consult the flights assigned to you. It also have access to the About page.
  2. Register:Only AdminOps can register a new account.
  3. View Flights: On the home page, you can see all scheduled flights.
  4. Add a New Flight: If you're an admin, you can add new flights to the schedule.
  5. Edit or Delete Flights: Admins can also edit or delete existing flights as needed.
  6. Check Dispatch: Admins have access to the dispatch page for additional operations.
  7. Profile Management: Users can view and edit their profile information.
