Welcome to Daily Flight Planner!

Daily Flight Planner is an innovative application designed to digitise and streamline the daily dispatch planning process for airports. Our platform offers a user-friendly interface that enhances efficiency and accuracy in managing flight operations.

  1. Log In: There are two types of logins: AdminOps and Dispatcher, each with access to different functionalities. To access, you need a login created by AdminOps.
    • AdminOps: It is a privileged access. It can access the profile currently in use, access the home page, search the database, edit and delete flights. It can access the about section. It has access to the dispatch section where it can modify and delete dispatchers. It has access to the register section where it can create a new account. It can access the new flight section where it can create a new flight.
    • Dispatcher: It can access the Profile page to view the currently logged-in profile. It can access the Home page to consult the flights assigned to you. It also has access to the About page and can access create, read, update and delete the report.
    • Glossary
    • P: Where the aircraft is parked
    • STAND: Where the aircraft arrival on stand
    • STA: Schedule Time od Arrival
    • STD: Schedule Time od Arrival
    • ATD: Actual Time of Departure
    • NIL: None or I have nothing to send to you
    • SPECIAL ASSISTANCE:WCHR(can walk and make steps), WCHS(can walk but not make steps)
    • INT: Internationa arrival
    • D: Domestical arrival
    • CTA: Republic of Irelad and Channes Island arrival
