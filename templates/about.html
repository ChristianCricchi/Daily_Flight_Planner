Welcome to Daily Flight Planner!

Here's some information about our app:

Daily Flight Planner, an innovative application designed to digitise and streamline the daily dispatch planning process for airports. Our platform offers a user-friendly interface that enhances efficiency and accuracy in managing flight operations.

  1. Log In or Register: Start by logging in to your account. If you don't have one, you'll need to register first.
  2. View Flights: On the home page, you can see all scheduled flights.
  3. Add a New Flight: If you're an admin, you can add new flights to the schedule.
  4. Edit or Delete Flights: Admins can also edit or delete existing flights as needed.
  5. Check Dispatch: Admins have access to the dispatch page for additional operations.
  6. Profile Management: Users can view and edit their profile information.
